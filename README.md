# Solana SPL Toolkit

An expert-level repository providing a clean, flat-structured interface for interacting with the Solana Program Library (SPL). 

## Features
* **Token Creation:** Streamlined scripts for creating Fungible Tokens.
* **Account Management:** Logic for creating Associated Token Accounts (ATA).
* **Minting & Transferring:** Robust functions for managing supply and P2P transfers.
* **Metadata Ready:** Compatible with Metaplex standards for token branding.

## Prerequisites
* Node.js v18.0 or higher
* A Solana wallet with devnet SOL (use `solana airdrop 2`)

## Usage
1. `npm install`
2. Configure your RPC in `config.js`
3. Run `node create-token.js`
